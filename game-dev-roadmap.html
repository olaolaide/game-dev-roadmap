Algorithm Modules
🔀
Sorting & Searching Intermediate

Essential for render ordering, AI priority, and leaderboard systems.

  • QuickSort, MergeSort, HeapSort — when to use each
  • Radix sort for integers (depth sorting in 2D games)
  • Timsort and why C# uses it for List.Sort()
  • Binary search on sorted arrays
  • Order statistics (kth smallest for LOD)
  • Partial sort — top-N enemies by threat level
  • Stable vs unstable sorts for layered rendering
🗺
Pathfinding Algorithms Advanced

The most critical algorithms for game AI movement.

  • BFS for shortest path on unweighted grids
  • Dijkstra's algorithm for weighted graphs
  • A* — heuristics (Manhattan, Euclidean, Octile)
  • Weighted A* and greedy best-first comparison
  • Jump Point Search for uniform-cost grids
  • Theta* for any-angle pathfinding
  • Flow field pathfinding for RTS crowd movement
  • Hierarchical pathfinding for large worlds
  • Path smoothing: funnel algorithm, Beziér
  • Dynamic obstacle avoidance (RVO/ORCA)
🌐
Spatial Data Structures Advanced

Efficient spatial queries are what separates fast games from slow ones.

  • Quadtree: insertion, deletion, range queries
  • Octree for 3D: splitting strategies
  • KD-tree for nearest-neighbor queries
  • BVH (Bounding Volume Hierarchy) for ray casting
  • Spatial hashing for dynamic objects
  • R-tree for rectangle overlap queries
  • Frustum culling math and implementation
  • BSP trees for indoor level geometry
  • Sweep and prune broad phase
🎲
Procedural Generation Advanced

Algorithmic content generation for infinite or emergent game worlds.

  • Noise functions: Perlin, Simplex, Value, Voronoi
  • Fractal Brownian Motion (fBm) for terrain
  • Diamond-Square algorithm for heightmaps
  • BSP dungeon generation
  • Cellular Automata for cave systems
  • Drunkard's Walk and room placement
  • Wave Function Collapse (WFC) algorithm
  • L-Systems for organic structures and flora
  • Delaunay triangulation for navmesh generation
  • Seeded RNG with PCG / Xoshiro256**
🤖
Game AI Foundations Advanced

Classic and modern AI techniques for believable NPC behavior.

  • Finite State Machines (FSM) — design and implementation
  • Hierarchical FSM (HFSM) and UML state charts
  • Behavior Trees: sequence, selector, parallel nodes
  • Utility AI — scoring systems for decision making
  • GOAP (Goal Oriented Action Planning)
  • Influence maps and tactical AI
  • Steering behaviors: seek, flee, arrive, pursue, flock
  • Reynolds flocking (separation, alignment, cohesion)
  • Minimax algorithm with alpha-beta pruning
  • Monte Carlo Tree Search (MCTS) for games
📐
Computational Geometry Advanced

Geometry algorithms powering collision, visibility, and mesh operations.

  • GJK collision detection algorithm
  • EPA (Expanding Polytope Algorithm)
  • Separating Axis Theorem (SAT)
  • Convex hull: Graham scan, Jarvis march
  • Polygon triangulation for mesh generation
  • Ray-triangle intersection (Möller–Trumbore)
  • Line-line and line-plane intersections
  • Minkowski sum for obstacle dilation
  • Visibility graph for 2D pathfinding
  • Ear-clipping triangulation
